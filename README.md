## S° (Sound Degree) & L° (Lumen Degree)

BEA_Core includes frequency resonance mapping functions inspired by how CD-ROM lasers scan physical media:

### Sound Degree (S°)
Maps audio frequencies to musical scale positions:

```
S° = log₂(f_audio / 440Hz) × 12
```

- 440 Hz (A4) → S° = 0°
- 880 Hz (A5) → S° = 12° (one octave up)
- 220 Hz (A3) → S° = -12° (one octave down)

### Lumen Degree (L°)
Maps light wavelengths to color wheel positions:

```
L° = (λ_visible / 555nm) × 360°
```

- 555nm (green, peak eye sensitivity) → L° = 0°
- 380nm (violet) → L° = ~246°
- 740nm (red) → L° = ~480°
